# BlazorRealTime
Real-time apps samples using Blazor and SignalR core 


## Blazor chat
This is a simple chat with 2 inputs, User and Message. it shows the status of the connection too. 

1. Compile the project
2. Run BlazorChat.Server
3. Run BlazorChat
